# CommonPlots

Overview
--------

ggutils contains utility functions for [ggplot2](http://ggplot2.org/) that retrieve information from, remove components of, and add text
 to a plot object created by 'ggplot2'.

Installation
------------
```r
# Install from GitHub:
# install.packages("devtools")
devtools::install_github('statech/ggutils')
```
